HTML5 Engine:
Tutorial Guide

Overview

The HTML5 Game Engine is an API written in HTML, JavaScript and WebGL that allows for the fast deployment of 2D games. This library is designed specifically for computer science students and faculty with no background in computer graphics or video games. This guide consists of a collection of tutorials demonstrating the basics of how to work with the HTML5 Game Engine. You can download a bundle of the files here.
